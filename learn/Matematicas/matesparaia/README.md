# Introducción

* 250 a.C Ctesibio de Alejandria
  * Primera maquina autocontrolada (molino)
* 1843 Ada Lovelace
  * Primer algoritmo
* 1936 Alan Turing
  * Primera calculadora
* 1959 Marvin Minsky y J.Mc. Carthy
  * Fundan el laboratorio de inteligencia artificial en el MIT
